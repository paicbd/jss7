Name
	m3ua get statisticsdelay

SYNOPSIS
	m3ua get statisticsdelay stackname <stack-name>

DESCRIPTION
	Returns the Statistics Delay value set

PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed.
					If not passed, the first stack configured in ShellExecutor
					will be used.

EXAMPLES
	m3ua get statisticsdelay
