
Name
	m3ua get statisticsperiod

SYNOPSIS
	m3ua get statisticsperiod stackname <stack-name>

DESCRIPTION
	Returns the Statistics Task Period value set

PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed.
					If not passed, the first stack configured in ShellExecutor
					will be used.

EXAMPLES
	m3ua get statisticsperiod
