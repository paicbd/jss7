Name
	m3ua error show

SYNOPSIS
	m3ua error show

DESCRIPTION
	This command is used to view the configuration of M3UA error handling.
    The information displayed will list all the errors configured that include
    the error name, the error code and the retry scheme. When the retry scheme is
    -1, it indicates an infinity retry.
